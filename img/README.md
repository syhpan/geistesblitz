# Encoding

## Corresponding numbers
1. Ghost - White
2. Couch - Red
3. Bottle - Green
4. Book - Blue
5. Rat - Gray

## File naming rules
|First|Second|Third|Fourth|
|-|-|-|-|
|Object1|Color1|Object2|Color2|
